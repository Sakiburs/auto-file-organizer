# Auto File Organizer

## 📌 Description
Auto File Organizer is a simple and smart Python desktop application that automatically organizes files into folders based on their type.

## 🚀 Features
- Select any folder (Downloads/Desktop/etc)
- Automatically sort files into categories
- Safe file handling (no deletion)
- Clean and beginner-friendly GUI

## 🛠 Technologies Used
- Python
- Tkinter
- OS Module

## ▶️ How to Run
1. Download or clone this repository
2. Run the app:
auto_file_organizer.py
